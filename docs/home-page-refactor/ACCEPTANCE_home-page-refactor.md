## 2. 功能验收详情

### 2.1 简历管理入口
- **需求**: 顶部导航增加"简历管理"，支持下拉菜单。
- **实现**: 
  - 修改 `Header.tsx`，添加 MouseEnter/MouseLeave 事件处理。
  - 下拉菜单包含"新建简历"和"我的简历"。
  - 选中状态高亮逻辑适配。

### 2.2 多简历支持
- **需求**: 用户可以创建、复制、修改多份简历。
- **实现**:
  - 新增 `/resumes` 页面，展示简历网格列表。
  - 实现复制功能：获取详情 -> 清除 ID/时间 -> 插入新记录 (Title + "副本")。
  - 实现编辑功能：路由 `/resume/edit/:id`，复用 `ResumeCreate` 组件，自动加载数据。

### 2.3 仪表盘更新
- **需求**: 显示最近 5 份简历。
- **实现**:
  - `Dashboard.tsx` 使用 `getResumeList(1, 5)` 获取数据。
  - 复用 `ResumeCard` 组件展示。
