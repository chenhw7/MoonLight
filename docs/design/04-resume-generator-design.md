# 简历生成功能设计文档

## 1. 功能概述

### 1.1 目标
提供表单填写式的简历生成功能，支持校招和社招两种模式，用户可通过标签页方式填写完整的简历信息。

### 1.2 用户场景
- **校招用户**：应届毕业生，需要填写教育经历、实习经历、项目经历、获奖情况等
- **社招用户**：有工作经验的求职者，需要填写工作经历、项目经历、技能特长等

### 2.2 招聘类型切换

页面顶部提供校招/社招模式切换按钮：
- **校招模式**：适合应届毕业生，强调教育背景、实习经历、获奖情况
- **社招模式**：适合有工作经验者，强调工作经历、项目成果

**切换行为**：
- 保留通用字段数据（基本信息、项目经历、技能等）
- 动态显示/隐藏特定字段（工作年限、当前公司等）
- 提示用户哪些字段将发生变化

### 2.3 标签页设计

| 标签页 | 校招显示内容 | 社招显示内容 |
|--------|-------------|-------------|
| **基本信息** | 姓名、手机号、邮箱、居住地、期望城市、求职状态、期望岗位 | 同校招 + 工作年限、当前公司、当前职位、期望薪资 |
| **教育经历** | 学校、专业、学历、时间、GPA、主修课程 | 学校、专业、学历、时间（GPA和课程可选） |
| **工作/实习** | 实习经历表单 | 工作经历表单 |
| **项目经历** | 项目表单 | 项目表单 |
| **技能与其他** | 技能特长、语言能力、获奖经历、作品展示、社交账号、自我评价 | 技能特长、获奖经历、作品展示、社交账号、自我评价（语言能力可选） |

## 4. 交互设计

### 4.1 表单操作

**添加记录**：
- 每个可重复模块（教育、工作、项目等）提供"+ 添加"按钮
- 点击后在列表下方新增空白表单
- 新表单自动展开，已填写表单可折叠

**删除记录**：
- 每条记录右上角提供删除按钮
- 删除前需二次确认

**排序**：
- 支持拖拽排序或上下箭头调整顺序
- 默认按时间倒序排列

### 4.2 导航操作

**标签切换**：
- 点击标签切换内容区域
- 自动保存当前标签页数据
- 未填写必填项时给出提示但不阻止切换

**上一步/下一步**：
- 按标签顺序导航
- 循环导航（最后标签的下一步回到第一标签）

**预览简历**：
- 点击后弹出简历预览模态框
- 实时渲染当前填写的数据
- 支持导出PDF

### 4.3 自动保存

- 每30秒自动保存草稿到本地
- 离开页面时提示保存
- 手动保存按钮随时可用

## 5. 校验规则

### 5.1 基本信息校验

| 字段 | 校验规则 |
|------|----------|
| 姓名 | 2-20个字符，仅中文或英文 |
| 手机号 | 中国大陆手机号格式 |
| 邮箱 | 标准邮箱格式 |
| 工作年限 | 正整数，0-50 |

### 5.2 时间校验

- 结束时间不能早于开始时间
- 支持"至今"选项（结束时间为空）
- 教育经历时间不能重叠

### 5.3 必填校验

- 提交时校验所有必填项
- 错误项高亮显示并滚动到可视区域
- 显示具体错误提示信息
