# 简历模板优化项目说明文档

## 一、项目规划

### 1.1 项目背景
当前简历模板存在视觉层次不清晰、信息密度不合理、排版缺乏专业感等问题，需要进行全面优化以提升简历的专业性和可读性。

### 1.2 项目目标
- 优化简历模板的视觉层次和排版
- 提升信息密度和空间利用率
- 增强专业感和可读性
- 适配A4尺寸打印需求

### 1.3 技术栈
- React + TypeScript
- Tailwind CSS
- 现有项目架构

---

## 二、实施方案

### 2.1 现有模板问题分析

| 问题项 | 当前状态 | 影响 |
|--------|----------|------|
| 页面边距 | 20mm | 内容区域被压缩 |
| 姓名标题 | 24px普通字重 | 视觉冲击力不足 |
| 联系信息 | 分散排列 | 占用空间过多 |
| 分区标题 | 普通下划线 | 专业感不足 |
| 时间格式 | 不统一 | 视觉混乱 |
| 技能展示 | 标签式 | 占用空间过多 |

### 2.2 计划一：经典专业版（推荐首选）

#### 设计理念
参考经典分区布局，强调专业性和可读性，适合大多数求职场景。

#### 核心改进

| 改进项 | 当前状态 | 改进方案 |
|--------|----------|----------|
| **页面边距** | 20mm | 调整为15mm，增加内容区域 |
| **姓名标题** | 24px普通字重 | 28px粗体，居中显示 |
| **联系信息** | 分散排列 | 单行紧凑排列，使用 \| 分隔 |
| **分区标题** | 普通下划线 | 全大写+下划线+适当间距 |
| **时间格式** | 2024.01 | 统一为 2024.01 格式 |
| **行距** | 默认 | 1.4-1.5倍行距，提升可读性 |
| **技能展示** | 标签式 | 改为文字描述式，更专业 |

#### 布局结构

```
┌─────────────────────────────────────────┐
│              姓 名 (28px 粗体)            │
│    电话 | 邮箱 | 地址 (一行紧凑排列)       │
├─────────────────────────────────────────┤
│  EDUCATION                              │
│  学校 · 专业 · 学历          时间        │
│  GPA/荣誉 (可选)                         │
├─────────────────────────────────────────┤
│  WORK EXPERIENCE / INTERNSHIP           │
│  公司 · 职位                 时间        │
│  • 工作职责1                            │
│  • 工作职责2                            │
├─────────────────────────────────────────┤
│  PROJECT EXPERIENCE                     │
│  项目名 · 角色               时间        │
│  • 项目描述                             │
├─────────────────────────────────────────┤
│  SKILLS                                 │
│  技能名: 熟练程度 | 技能名: 熟练程度      │
├─────────────────────────────────────────┤
│  LANGUAGES                              │
│  语言: 熟练程度 | 语言: 熟练程度          │
├─────────────────────────────────────────┤
│  AWARDS & HONORS                        │
│  • 获奖名称 (时间)                       │
├─────────────────────────────────────────┤
│  PORTFOLIO / SOCIAL LINKS / SUMMARY     │
└─────────────────────────────────────────┘
```

#### 样式规范

**字体层级**
- 姓名: 28px, font-bold, color: #1a1a1a
- 分区标题: 12px, font-bold, uppercase, letter-spacing: 0.1em
- 学校/公司名: 13px, font-semibold
- 职位/专业: 正常字体大小
- 描述文字: 11px, normal, line-height: 1.5

**间距规范**
- 页面边距: 15mm
- 分区间距: 16px
- 条目间距: 10-12px
- 行高: 1.5

**颜色规范**
- 主文字: #1a1a1a
- 次要文字: #4a4a4a
- 辅助文字: #666666
- 分割线: #333333 (2px solid)

### 2.3 计划二：现代简洁版（带照片支持）

#### 设计理念
采用顶部集中式布局，适合展示个人形象，强调现代感和专业性。

#### 核心改进

| 改进项 | 改进方案 |
|--------|----------|
| **照片位置** | 右上角固定位置，根据比例显示不同尺寸 |
| **头部区域** | 左侧姓名+联系信息，右侧照片，形成视觉平衡 |
| **色彩运用** | 标题使用深灰色(#333)，分割线使用浅灰(#ddd) |
| **字体层级** | 姓名>公司学校>职位专业>描述内容，形成4级层级 |
| **图标使用** | 电话、邮箱、地址使用emoji图标，提升现代感 |

#### 布局结构

```
┌─────────────────────────────────────────┐
│  姓 名 (左)                    ┌─────┐  │
│  📱电话 ✉️邮箱 📍地址 (左)      │照片 │  │
│                                └─────┘  │
├─────────────────────────────────────────┤
│  EDUCATION                              │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  学校                        时间        │
│  专业 · 学位                            │
│  GPA/荣誉 (可选)                        │
├─────────────────────────────────────────┤
│  WORK EXPERIENCE                        │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  公司 · 职位                 时间段      │
│  • 工作职责描述                         │
├─────────────────────────────────────────┤
│  PROJECTS                               │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │

│  SKILLS | LANGUAGES | AWARDS            │

│  分类展示，每类一行                      │
```

**间距规范**

**颜色规范**

### 4.2 关键修改点

1. **ResumePreview.tsx**: 
   - 添加模板切换功能（按钮切换）
   - 支持两种模板动态切换
   - 根据模板类型调整页面边距

2. **ClassicTemplate.tsx**: 
   - 实现经典专业版样式
   - 姓名居中显示
   - 联系信息单行紧凑排列
   - 分区标题全大写+粗下划线（中文标题）

3. **ModernTemplate.tsx**: 
   - 实现现代简洁版样式
   - 头部左右布局（信息+照片）
   - 联系信息纯文字显示（无emoji）
   - 细分割线设计（中文标题）

### 4.3 模板切换功能

在预览组件顶部工具栏添加了模板切换下拉框：

## 六、使用说明

### 6.1 如何切换模板

1. 进入简历预览页面
2. 在顶部工具栏找到模板切换下拉框
3. 选择"经典专业版"或"现代简洁版"
4. 预览区域会实时更新显示效果

### 6.2 模板选择建议

- **经典专业版**：适合传统行业、金融、法律等专业岗位
- **现代简洁版**：适合互联网、创意行业、设计等岗位
